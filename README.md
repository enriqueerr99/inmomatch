## 📋 Descripción

**InmoMatch** es un curso que enseña a emprendedores cómo convertirse en **"Matchers"** — profesionales que conectan inversores globales con proyectos inmobiliarios de construcción en Latinoamérica.

No es un curso de real estate tradicional. Es un modelo de negocio nuevo:

- ✅ Conecta oferta (proyectos) con demanda (inversores)
- ✅ Gana 4-5% de comisión por venta
- ✅ Transacciones de $100k-$400k USD
- ✅ 2-4 ventas/mes es alcanzable
- ✅ Ingreso: €8k-€30k/mes (Año 1 realista)

---

## 🎬 La Profesión: Matcher

### ¿Qué es un Matcher?

Un **Matcher** es un intermediario estratégico que:

1. **Identifica** proyectos de construcción reales en LATAM
2. **Valida** comisiones con desarrolladores (4-5%)
3. **Atrae** inversores internacionales vía Meta Ads
4. **Cierra** ventas de $100k-$400k USD
5. **Gana** 4-5% de comisión por cada venta

### Por qué funciona:

| Aspecto | Diferencia |
|---------|-----------|
| **vs Agente tradicional** | Mejor comisión (4-5% vs 3-4%) + Volumen (2-4 vs 0.5-1.5 ventas/mes) |
| **vs Affiliate** | Transacciones reales + Profesión propia + Identidad clara |
| **vs Desarrollador** | No tienes que construir + Acceso a inversores globales + Sin riesgo |

### Ingresos Realistas (Año 1):

```
Mes 1-3:  €4.5k-€17k/mes neto
Mes 4-6:  €10k-€22.5k/mes neto
Mes 7-12: €15k-€30k/mes neto

TOTAL AÑO 1: €70k-€150k neto
```

## 💰 El Modelo de Negocio

### 4 Fases del Matcher:

#### 1️⃣ SOURCING
```
Actividad: Buscar y validar proyectos
Timeline: 2-4 semanas por proyecto
Output: 1-2 proyectos confirmados con comisiones
```

#### 2️⃣ MARKETING
```
Actividad: Atraer inversores con Meta Ads
Budget: €5k-€15k/mes
Output: 20-50 leads/mes calificados
```

#### 3️⃣ CLOSING
```
Actividad: Presentar y cerrar ventas
Conversion: 2-5% de leads
Output: 2-4 ventas/mes
```

#### 4️⃣ MONETIZAR
```
Ingreso: 4-5% de comisión
Por venta: €4k-€20k
Mensual: €8k-€80k bruto
```

### Ejemplo Real:

```
📍 México - Proyecto residencial
├─ Valor por unidad: $250,000 USD
├─ Comisión: 5% = $12,500 USD (€11,500)
│
├─ Escenario Realista (3 ventas/mes)
│  ├─ Ingreso bruto: $37,500/mes (€34.5k)
│  ├─ Gastos (ads, taxes, etc): €10k
│  └─ Ingreso neto: €24.5k/mes
│
└─ Año 1: €294k bruto → €180k neto
```

### Lo que Funciona:
✅ El modelo es real y viable  
✅ 4-5% comisión es realista  
✅ 2-4 ventas/mes es alcanzable

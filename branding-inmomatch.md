## 🎯 PALETA DE COLORES

### Colores Principales:

```
Azul Oscuro (Primario)
#1e2c41
RGB: 30, 44, 65
Uso: Headers, botones principales, fondos oscuros

Amarillo (Secundario/Accent)
#fed427
RGB: 254, 212, 39
Uso: Acentos, destacados, CTAs, highlights

Blanco (Fondo)
#ffffff
RGB: 255, 255, 255
Uso: Fondo, texto sobre colores oscuros
```

### Paleta Extendida (Recomendada):
- **Gris claro:** #f5f5f5 (fondos secundarios)
- **Gris medio:** #cccccc (bordes, dividers)
- **Gris oscuro:** #666666 (texto secundario)
- **Negro:** #000000 (texto principal, contraste)

## 🔤 TIPOGRAFÍA

**Font Principal:** Poppins (Google Fonts - Libre)

### Jerarquía de Textos:

#### Títulos (H1)
- Font: Poppins Bold (700)
- Size: 48px - 56px
- Color: #1e2c41 (Azul Oscuro)
- Line-height: 1.2
- Ejemplo: "Conviértete en un Matcher"

#### Subtítulos (H2)
- Font: Poppins SemiBold (600)
- Size: 32px - 40px
- Color: #1e2c41 o #fed427
- Line-height: 1.3
- Ejemplo: "Gana €4k-€20k por venta"

#### Títulos de Sección (H3)
- Font: Poppins SemiBold (600)
- Size: 24px - 28px
- Color: #1e2c41
- Line-height: 1.4
- Ejemplo: "Módulo 1: Qué es un Matcher"

#### Texto Corporal
- Font: Poppins Regular (400)
- Size: 16px
- Color: #333333
- Line-height: 1.6
- Spacing: 1.5rem entre párrafos
- Ejemplo: Párrafos largos, descripciones

#### Citas/Testimonios
- Font: Poppins Italic (400)
- Size: 18px
- Color: #fed427 (accent) o #1e2c41
- Line-height: 1.6
- Ejemplo: "Los Matchers hacen matching internacional"

#### Pie de Foto/Meta
- Font: Poppins Regular (400)
- Size: 12px
- Color: #999999
- Style: Italic
- Ejemplo: "Ejemplo de un Matcher cerrando una venta"

## ✨ PRINCIPIOS DE DISEÑO

1. **Minimalista:** Usa mucho espacio en blanco
2. **Contraste fuerte:** #1e2c41 sobre #ffffff y viceversa
3. **Amarillo como sorpresa:** Úsalo estratégicamente para CTAs
4. **Tipografía limpia:** Poppins es moderna y confiable
5. **Accesibilidad:** Mantén contraste alto (WCAG AA mínimo)
6. **Coherencia:** Mantén la paleta en todos los canales

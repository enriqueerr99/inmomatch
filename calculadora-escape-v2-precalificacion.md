# 🧮 CALCULADORA DEL ESCAPE
## Simple, Cercana y Directa

---

## 💭 LA IDEA

No es un formulario árido. Es una conversación.

Como si te preguntara un colega tomar café que de verdad quiere entender tu situación.

**Objetivo:** Entiende dónde estás, qué quieres, si es realista, y qué le prometes.

---

## 📝 LAS PREGUNTAS (13 PREGUNTAS TOTALES)

### PARTE 1: DÓNDE ESTÁS AHORA (5 preguntas)

**P1: Hola, ¿cuál es tu nombre?**
```
Input texto
Vibe: Cercano, personal
```

**P2: ¿En qué ciudad o región vives?**
```
Input texto (o dropdown de ciudades españolas)
Vibe: Simple, sin clasificar
```

**P3: ¿Cuál es tu trabajo actual?**
```
Input texto libre
Ej: "Programador en empresa", "Vendedor", "Abogado", etc
```

**P4: ¿Cuánto ganas al mes (neto)?**
```
Input número: €500-€10,000
Vibe: Sin juzgar, es solo un número
```

**P5: ¿Cuántos ahorros tienes ahora?**
```
Input número: €0-€100,000
Vibe: "Para saber si tienes colchón para empezar"
```

---

### PARTE 2: TIEMPO Y REALIDAD (3 preguntas)

**P6: ¿Cuántas horas a la semana tienes LIBRES reales?**
```
Slider o número: 0-40 horas
Vibe: "Serio, no te juzgo si son 2 horas, solo necesito saber"
```

**P7: ¿En qué momento tienes ese tiempo libre?**
```
Checkboxes (multiselect):
- Mañanas (6-9am)
- Mediodías (12-14h)
- Tardes (18-21h)
- Noches (21-24h)
- Fines de semana

Vibe: Casual, sin juicio
```

**P8: ¿Has probado algún otro negocio antes?**
```
Radio buttons:
- No, sería mi primer intento
- Sí, pero no funcionó
- Sí, y me fue bastante bien
- Sí, pero nunca llegué a cerrar

Vibe: Entender si ya conoces el juego o no
```

---

### PARTE 3: LO QUE QUIERES (3 preguntas)

**P9: ¿Cuándo te gustaría dejar tu trabajo?**
```
Radio buttons:
- Ya, estoy quemado
- En 3 meses
- En 6 meses
- En 1 año
- Sin prisa, cuando sea

Vibe: Sin presión, solo saber dónde estás
```

**P10: ¿Te gustaría trabajar desde donde quisieras, sin horarios fijos, solo con un portátil?**
```
Radio buttons:
- Sí, eso sería increíble
- Sí, pero necesitaría algo de estructura
- No, prefiero horario fijo
- Me da igual

Vibe: Aspiracional pero real
```

**P11: ¿Cuál es tu sueño REAL si pudieras tener libertad?**
```
Text area largo (opcional)
Ej: "Viajar 3 meses al año", "Trabajar desde una playa", "Pasar más tiempo con mi familia"

Vibe: Emocional, lo que REALMENTE quieres
```

---

### PARTE 4: LA OFERTA (2 preguntas)

**P12: ¿Te parece razonable ganar €5,000-€15,000 por venta?**
```
Radio buttons:
- Sí, suena increíble
- Sí, pero suena difícil
- Quizá, no sé si sea realista
- No, me parece mucho

Vibe: Calibrar expectativas
```

**P13: Si todo esto fuera posible, ¿te interesaría aprender cómo hacerlo?**
```
Radio buttons:
- Sí, totalmente
- Quizá, depende del precio/contenido
- No sé, necesito más info
- No, no me interesa

Vibe: Cierre, saber si hay intención real
```

## 📊 SEGMENTACIÓN SIMPLE (Detrás de escenas)

```
JAVIER ES HOT LEAD SI:
├─ Trabaja hace 3+ años (está atrapado)
├─ Gana €1,200-€3,000/mes
├─ Tiene €5,000+ ahorros
├─ Disponible 6+ horas/semana
├─ Quiere salir en 6 meses
└─ Le interesa la libertad geográfica

JAVIER ES WARM LEAD SI:
├─ Tiene 1-2 de los anteriores

JAVIER NO ES CLIENTE SI:
├─ Menos de 3 horas/semana libres
├─ Sin ahorros (no puede pagar €1,690)
├─ "Sin prisa" (no es urgente)
└─ No le atrae libertad/viajes
```
